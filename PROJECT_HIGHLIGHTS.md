# 项目深度总结：Yuan 内容管理与安全交互系统

## 1. 项目概览
**Yuan** 是一个基于 **Vue 3 + Koa 2 + MongoDB** 的全栈内容管理系统。项目采用前后端分离架构，旨在通过现代化的前端设计与严谨的后端安全逻辑，提供一个可扩展、易运维、且具备高度数据安全性的交互平台。

---

## 2. 全栈技术细节

### 前端架构 (Vue 3 生态)
- **核心框架**: Vue 3.2+ (Composition API + Script Setup)，利用响应式系统提升组件更新性能。
- **状态管理**: **Pinia** + **pinia-plugin-persistedstate**，实现用户状态的全局同步与跨会话持久化。
- **构建工具**: **Vite**，通过原生 ESM 特性实现毫秒级热更新（HMR）。
- **UI 体系**: **Naive UI** + **LESS**，采用全组件按需加载策略，极致优化首屏加载性能。
- **通信与安全**: **Axios** 深度封装 + **Web Crypto API**。前端不存储静态公钥，而是通过后端 API 动态获取公钥进行 RSA-OAEP-256 加密。

### 后端架构 (Koa 2 生态)
- **核心引擎**: **Koa 2**，采用洋葱模型中间件处理请求生命周期，利用异步流控制实现高性能并发处理。
- **数据库**: **MongoDB** + **Mongoose**，通过 Schema 模式化非关系型数据，并支持复杂的数据聚合查询。
- **认证与鉴权**: **JWT (jsonwebtoken)** + **Bcrypt**。密码存储采用高强度加盐哈希，传输层采用 RSA 非对称解密。
- **配置与安全**: 实现了 **Key-Manager 自动初始化模块**，在容器启动时动态生成 2048 位 RSA 密钥。

---

## 3. 核心技术亮点

### 🌟 亮点一：基于动态 RSA 的全链路安全认证体系
*   **技术实现**: 后端动态生成 RSA 密钥对，并暴露受控接口。前端在登录前实时拉取。改用浏览器原生 `window.crypto.subtle` API 实现 RSA-OAEP 加密。
*   **价值**: 彻底杜绝了私钥在 Git 仓库或构建产物中泄露的风险。

### 🌟 亮点二：容器化环境下的状态持久化与配置分离
*   **技术实现**: 使用 `process.cwd()` 实现路径自适应。通过 Docker Compose Volumes 将 `./config` 与 `./mongodb-data` 持久化到宿主机。
*   **价值**: 实现了“镜像即安全”的闭环，让系统具备了极强的迁移能力。

### 🌟 亮点三：自动化运维与定制化构建工具链
*   **技术实现**: 自研 `pack.js` 工具链，集成了环境自检、代码混淆、自动化脚本生成。
*   **价值**: 将繁琐的打包部署流程缩短至 1 分钟内，体现了极高的工程化管理价值。
